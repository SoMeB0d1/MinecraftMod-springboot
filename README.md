# TFC-SpringBoots

TFC-SpringBoots 是一个 Minecraft mod，提供与 TFC（TerraFirmaCraft）主题相关的功能与资源。

- **Mod 名称**：TFC-SpringBoots
- **查询地址**：可在 CurseForge 上搜索并查看该 mod 的更多信息。

## 概述
本 mod 为 Minecraft 提供若干扩展功能（以 TFC 风格为主）。在独立的 Minecraft 实例中经验证，该 mod 能够正常运行。

## 开发环境说明（重要）
- 在本项目的开发环境中，前置 mod `Patchouli` 无法正常运行，可能导致基于 Patchouli 的手册或文档在开发调试时不可用或行为异常。
- 尽管开发环境中 Patchouli 有问题，但在额外的（独立的）Minecraft 实例中，将本 mod 与相同的依赖一起运行时，mod 本身能够正常工作。

如果你在开发时依赖 Patchouli 的手册内容进行调试或展示，建议在单独的游戏实例中进行运行与测试，或在运行时使用已知能正常工作的 Minecraft 客户端实例来验证功能。

## 使用与安装
1. 将构建好的 jar 文件放入 Minecraft 的 `mods` 目录。
2. 启动带有对应 Forge/版本的 Minecraft 客户端。
